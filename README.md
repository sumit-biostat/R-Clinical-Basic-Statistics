# R Clinical Basic Statistics

This repository contains R scripts to perform basic statistical analysis on Alzheimer's clinical data.

## Features
- Descriptive statistics: mean, median, SD
- Frequency tables & proportions
- Inferential statistics: t-tests, chi-square, ANOVA
- Visualizations: histograms, boxplots

## Folder Structure
- `data/` - contains input CSV files
- `scripts/` - R scripts for analysis
- `outputs/` - optional folder to save plots or results
